# Audio Assets

Place royalty-free audio clips (`.mp3`, `.wav`, `.aac`, `.m4a`) in this directory.

The AudioAgent will randomly select a file and merge it into the video at ~8% volume as background music.

## Naming Convention

Optionally name files with category prefixes for smarter selection:
- `tech_upbeat.mp3` → selected when topic category is "tech"
- `sports_energetic.mp3` → selected for "sports" topics
- `general_chill.mp3` → selected for "general" topics

## Free Sources for Audio

- [Pixabay Music](https://pixabay.com/music/) — free, no attribution
- [Mixkit](https://mixkit.co/free-stock-music/) — free stock music
- [Free Music Archive](https://freemusicarchive.org/) — CC-licensed
- [YouTube Audio Library](https://studio.youtube.com/channel/UC/music) — free for creators
