# Font Assets

Place custom `.ttf` or `.otf` font files in this directory for use in video text overlays.

If no fonts are placed here, the system uses the default system font (DejaVu Sans on Ubuntu/GitHub Actions).

## Recommended Free Fonts

- [Inter](https://fonts.google.com/specimen/Inter) — clean, modern
- [Outfit](https://fonts.google.com/specimen/Outfit) — bold, geometric
- [Poppins](https://fonts.google.com/specimen/Poppins) — friendly, rounded
- [Montserrat](https://fonts.google.com/specimen/Montserrat) — elegant, professional

Download from [Google Fonts](https://fonts.google.com/) and place the `.ttf` files here.
